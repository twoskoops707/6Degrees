## People Data Labs

### Overview
People Data Labs provides comprehensive professional profiles and contact information for individuals.

### Application Process
1. Visit [People Data Labs](https://www.peopledatalabs.com/)
2. Click "Get Started" or "Sign Up"
3. Fill out the registration form with your information
4. Verify your email address
5. Access your API key in the dashboard

### API Limits
- Free tier: 1,000 requests per month
- Paid tiers available for higher usage

### Documentation
[People Data Labs API Documentation](https://docs.peopledatalabs.com/)

## Clearbit

### Overview
Clearbit provides business intelligence and company data including enrichment and discovery services.

### Application Process
1. Visit [Clearbit](https://clearbit.com/)
2. Click "Sign Up" or "Get API Access"
3. Fill out the registration form
4. Verify your email address
5. Access your API key in the dashboard

### API Limits
- Free tier: Limited access with basic features
- Paid tiers offer higher rate limits and additional features

### Documentation
[Clearbit API Documentation](https://clearbit.com/docs)

## Pipl

### Overview
Pipl is an identity resolution service that aggregates data from public sources to create comprehensive profiles.

### Application Process
1. Visit [Pipl API](https://pipl.com/api/)
2. Click "Get API Key" or "Sign Up"
3. Register for an account with your details
4. Verify your email address
5. Find your API key in the account dashboard

### API Limits
- Free tier: Limited number of requests per month
- Paid plans available for increased usage

### Documentation
[Pipl API Documentation](https://docs.pipl.com/)

## BuiltWith

### Overview
BuiltWith provides technology profiling for websites and companies, identifying the tools and services they use.

### Application Process
1. Visit [BuiltWith](https://builtwith.com/)
2. Click "API" in the navigation menu
3. Choose the appropriate API plan
4. Register for an account
5. Receive your API key via email or in the dashboard

### API Limits
- Various pricing tiers based on request volume
- Free tier with limited requests available

### Documentation
[BuiltWith API Documentation](https://api.builtwith.com/)

## Hunter.io

### Overview
Hunter.io specializes in email discovery and verification services for professionals and businesses.

### Application Process
1. Visit [Hunter.io](https://hunter.io/)
2. Click "Get Started" or "Sign Up"
3. Create an account with your email address
4. Verify your email address
5. Access your API key in the dashboard under "API Keys"

### API Limits
- Free tier: 100 verifications/day and 50 domain searches/month
- Paid plans offer higher limits

### Documentation
[Hunter.io API Documentation](https://hunter.io/api)

## HaveIBeenPwned

### Overview
HaveIBeenPwned provides data breach exposure information to check if email addresses or passwords have been compromised.

### Application Process
1. Visit [HaveIBeenPwned API](https://haveibeenpwned.com/API/v3)
2. For personal use, no API key is required (rate limited)
3. For commercial use, API keys are available through payment
4. For enterprise usage, contact sales@haveibeenpwned.com

### API Limits
- Free usage: Rate limited to 1 request per 1.5 seconds
- Commercial API keys: Higher rate limits available
- Enterprise: Custom rate limits

### Documentation
[HaveIBeenPwned API Documentation](https://haveibeenpwned.com/API/v3)
