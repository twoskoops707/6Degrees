# 6Degrees OSINT Tool

An Android application for gathering publicly available information about people and companies through legal OSINT techniques.

## Features

- Person search by name, email, phone number, or photo
- Company intelligence gathering
- Property ownership records
- Vehicle registration information
- Financial summary reports
- Data breach exposure checking
- Comprehensive reporting with export capabilities

## Legal Compliance

This tool only accesses publicly available information through legal means:
- Official government databases
- Public records
- Social media profiles (public information only)
- Business registries
- Professional licensing boards

All data gathering respects:
- GDPR compliance
- CCPA compliance
- Terms of service of each data source
- Proper attribution and source documentation

## Installation

### From GitHub Releases
1. Go to [Releases](https://github.com/yourusername/6degrees-osint/releases)
2. Download the latest APK file
3. Install on your Android device

### Building from Source
1. Clone this repository
2. Open in Android Studio
3. Build and run

## API Keys

To use this application, you'll need API keys from:
- People Data Labs
- Clearbit
- Pipl
- BuiltWith
- Hunter.io
- HaveIBeenPwned

Add these keys in the Settings section of the app.
